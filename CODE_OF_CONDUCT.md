# Contributor Covenant Code of Conduct

## 我们的承诺

作为项目维护者和贡献者，我们承诺：

- 创建一个开放、友好、包容和尊重的社区；
- 允许不同背景和观点的人安全地参与。

## 可接受的行为示例

- 使用友善与包容的语言；
- 尊重不同的观点和经验；
- 乐于接受建设性的批评；
- 关注对社区最有利的事情；
- 对其他社区成员表现出同理心。

## 不可接受的行为示例

- 使用贬损、侮辱或攻击性言语；
- 发表任何形式的仇恨言论或骚扰；
- 公开或私下骚扰；
- 未经明确许可，公开他人隐私信息；
- 其他合理认定为不恰当的行为。

## 维护者的责任

项目维护者有权利和责任：

- 删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、Wiki 编辑、问题和其他贡献；
- 暂时或永久地禁止任何他们认为行为不适当、威胁、冒犯或有害的贡献者。

## 适用范围

本行为准则适用于：

- 项目空间内；
- 项目相关的公共空间（例如社交媒体、线下活动等）。

## 举报与联系方式

如你发现违反本行为准则的行为，可以：

- 通过 Issue 或私信联系项目维护者；
- 或使用仓库中提供的联系邮箱。

维护者将尽最大努力在不泄露举报人身份的前提下处理相关问题。
