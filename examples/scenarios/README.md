# 官方 Demo 场景 · Wuwei Psyche Causal Lab

这里收录了若干「典型心理场景」，用于演示：

- 文本 → PsycheVector 的映射
- 因果图与风险评分的直观效果
- 简化反事实（counterfactual）实验如何使用

> 注意：以下示例仅用于研究、教学和产品设计讨论，
> 不构成任何医疗或治疗建议。

---

## S1. 工作压力 × 自我怀疑

**示例文本：**

> 最近工作压力特别大，总觉得自己不够好，担心哪天就被淘汰了。加班到很晚也睡不好，一想到明天开会就心慌。

**预期结构特征：**

- 动因：安全感 / 价值感 较高
- 信念偏差：自我怀疑（self_doubt）、完美主义
- 情绪：焦虑、紧张
- 行为倾向：过度用力 / 回避开会
- 风险评分：中等偏上
- 人格分桶：safety_dominant 或 value_dominant（视实现而定）

---

## S2. 亲密关系不安

**示例文本：**

> 最近总觉得对象会离开我，他没回消息我就一直刷手机，会脑补各种最坏的情况，控制不住想发很多消息去确认。

**预期结构特征：**

- 动因：归属感 / 认同感 较高
- 信念偏差：被抛弃恐惧、低自我价值感
- 情绪：不安、忐忑、占有欲
- 行为倾向：黏附、过度确认
- 风险评分：中等
- 人格分桶：belonging_dominant / recognition_dominant

---

## S3. 轻度抑郁样无力感

**示例文本：**

> 最近什么都提不起兴趣，做什么都觉得没意义。明明也没有特别大的事，但就是觉得每天像机械一样活着，心里挺空的。

**预期结构特征：**

- 动因：价值感与认同感的能量下降
- 信念偏差：无意义感、习得性无助
- 情绪：空虚、轻度抑郁感、倦怠
- 行为倾向：拖延、回避、主动性降低
- 风险评分：中等或略偏上
- 策略倾向：温和激活、自我照顾、寻求现实支持，而不是「硬扛」

---

## S4. 好奇探索型（低风险）

**示例文本：**

> 最近对心理学和人工智能特别好奇，感觉像打开了一个新的世界，很想系统地学一学，看看能不能用在自己的人生和工作上。

**预期结构特征：**

- 动因：好奇心、成长与价值实现
- 信念偏差：较健康（成长型心态）
- 情绪：兴奋、期待
- 行为倾向：探索、学习、规划
- 风险评分：较低
- 人格分桶：curiosity_dominant / value_dominant

---

## 如何运行所有场景 Demo

可以使用脚本 `examples/scripts/demo_scenarios.py`：

```bash
python examples/scripts/demo_scenarios.py
```

它会依次：

1. 运行完整 pipeline，打印 PsycheVector 关键字段；
2. 打印人格分桶与风险评分；
3. 调用简化反事实引擎，对 `"self_doubt"` 做一次小幅度调整。
