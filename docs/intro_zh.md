# 施无畏心理因果实验室 · 说明文档（简体中文）

本仓库提供的是一套「动因 – 信念 – 情绪 – 行为」链条的**简化心理因果引擎（Lab Edition）**。

核心目标：

- 把心理结构变成**可计算的中间层**（PsycheVector）
- 用一个**可视化因果图**表达「为什么会这样想/这样做」
- 提供一个**简化的反事实实验接口**，用于问一句：「如果这个信念缓和一点，会怎样？」

## 一、模块概览

- `engine/vectorizer`：文本 → PsycheVector
- `engine/causal/simple_causal_engine.py`：简化因果图与反事实演示
- `engine/causal/advanced_placeholder.py`：高级因果引擎的占位符（不开源）
- `engine/persona`：人格分桶
- `engine/strategy`：低风险建议
- `engine/safety`：安全层
- `engine/runtime`：EngineCore 主调度

其中真正具有专利价值和商业价值的「高级反事实引擎（M3.4–M3.6）」不会出现在此仓库中。

## 二、适用对象

- 心理学 / 行为科学研究者
- 对话系统 / 情感计算 / 用户体验设计工程师
- 探索「类人智能架构」的开发者
- 对动因–信念–情绪–行为链有兴趣的好奇者

## 三、使用边界

请务必阅读仓库根目录下的：

- `DISCLAIMER.md`
- `ETHICS.md`
- `SECURITY.md`

开源版仅用于：

- 研究与教学
- 原型验证与体验优化
- 心理因果结构的可视化

明确禁止用于：

- 医疗诊断、治疗决策
- 对未成年人的闭环情感操控
- 成瘾性产品中的心理剥削优化
- 任何违法或高风险用途

## 四、与完整架构/专利的关系

本仓库是完整体系中的一个「可公开展示层」：

- 完整体系还包含：  
  - Ψ/Φ/Ξ 三域心智动力机  
  - 完整 M-B-A-C 四层结构  
  - 规则底座（外部世界规则）  
  - 更复杂的反事实推理与路径干预模块

- 本仓库仅暴露：
  - 一部分中间结构（PsycheVector）
  - 一个示意级因果图
  - 一个玩具级反事实 demo
  - 严格安全边界与伦理说明

这样做的目的是：

- 让研究和工程社群有一个可以直接运行和改造的「心理因果引擎原型」
- 同时不暴露关键商业实现与专利核心算法。
